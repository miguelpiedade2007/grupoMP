Primeiro projeto git do Checkpoint 1 da aula de Data structures and Algorithims do Professor Erick Toshio Yamamoto

O checkpoint consiste em criar um projeto em grupo em que criamos um arquivo main.c integrado entre os membros do grupo em que cada membro consiga fazer seu nome aparecer na tela.

Eu, Rafael, e o Miguel, fizemos a atividade em dupla e estruturamos o arquivo C com 3 printf, sendo o primeiro para título e os outros dois para os nomes e RMs dos alunos participantes.
